There are now two ways to install Macaulay2 under Mac OS X:
  • installing using homebrew
  • downloading a disk image
We describe both of them below.

Installing using homebrew

Mahrud Sayrafi has created a "homebrew tap" supporting many binary distributions of Macaulay2. To install it that way, follow the instructions at our homebrew tap.
